- Konten asli dimulai dengan pernyataan import, seperti `import { Map, Swords } from 'lucide-react'`
- Harap mempertahankan semua baris import ini di awal transkripsi
- Jangan lewati, ubah, atau terjemahkan apapun pernyataan import

**Peraturan 2：⛔️ Jangan menambah tanda kutip kode blok (```**)
- ⚠️ Ini adalah kesalahan yang paling umum, jangan lakukan!
- Jangan menambah tanda kutip kode blok di depan pernyataan import
- Jangan menambah tanda kutip kode blok di depan dan di belakang pernyataan metadata export
- Jangan menambah tanda kutip kode blok di konten MDX
- ❌ Contoh Kesalahan:
  ```
  ```
  import { Map } from 'lucide-react'
  ```
  Ini salah! Jangan lakukan demikian!

  ```
